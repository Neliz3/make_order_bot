## Setup

### Technologies Used
* Python 3.10
* Aiogram 3.3
* SQLAlchemy


### Installation
```python3 -m venv env```


```. env/bin/activate```


```pip install -r requirements.txt```

```python bot.py```


### Step-by-step installation

```bash
pip install -U aiogram
```
```bash
pip install python-dotenv
```
```bash
pip install SQLAlchemy
```
```bash
sudo apt-get -y install postgresql
```
